# Dynamic Planogram Engine

This repository contains the full 4-phase planogram engine:
- Phase 1: Scorer
- Phase 2: Allocator
- Phase 3: Placer
- Phase 4: Optimizer

Folders:
- backend/ — all engine logic
- frontend/ — visualizer
- docs/ — specifications (v2.1 heuristic engine)
- tests/fixtures — test datasets
