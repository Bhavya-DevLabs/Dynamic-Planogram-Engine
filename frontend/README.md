# Frontend (React) - Dynamic Planogram Engine

This folder contains all UI code for:
- SKU input forms
- Shelf configuration
- Visualization canvas
- Engine execution and result display

Structure:
- components/  → reusable UI blocks
- views/       → full screens (Phase 1, Phase 2, Phase 3, Phase 4)
- services/    → API calls to backend or AI engine
- types/       → TypeScript interfaces (mirroring backend data models)
